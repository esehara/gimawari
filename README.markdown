Gimawari
--------

Gimawariとは
------------

　Gimawariとは、Pythonで書かれたGitの日本語版です。名前の由来は、日本語でプログラミングできる「ひまわり」から頂きました。あと、Gitまわりのことをやってくれるという意味で、「Gitまわり -> Gimawari」という名前を付けました。

Gimawariの動作環境
------------------

　作者の環境はこちらです

* Python 2.7.1
* Ubuntu 11.10

Gimawariのインストール
----------------------

　このレポジトリをCloneして、 `python setup.py install` を実行してください。
